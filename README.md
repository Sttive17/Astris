# Astris

Astris is a React + Vite web experience focused on adaptive workplace matching and inclusive hiring flows.

## Structure

- src/app: application entry point and main UI
- src/components: reusable UI building blocks
- src/lib: API and shared helpers
- src/styles: global styles and theme layers
- public: static assets

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
